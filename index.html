SoundCloud scraper

This python package scrapes SoundCloud comments with a selection of built in and customisable filters. Data is exported to CSV, and can be scraped from individual track URLs, or parent URLs such as playlists or artist pages.

This project was intended initially for use in music scholarship, so I designed a Tkinter GUI as the main interface. However, the modules could be imported and used independently as in the example at the bottom of /src/app/backend/scraper.py.

This was my first ever coding project - excuse some poor design choices here and there! At some point I might revisit the code and refactor it for a simple web app. It might be a good project to practice some data analysis or ML - analysing themes or detecting spam in an artist's audience for example.

I'm aware I could have used SoundCloud's API for much better performance and reliability - I wanted to practice data scraping at this stage! Scraping also bypasses any API restrictions that might be in place.
